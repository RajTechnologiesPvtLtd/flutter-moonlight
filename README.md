## Run Locally

Clone the project

```bash
git clone https://github.com/RajTechnologiesPvtLtd/flutter-moonlight moonlight
```

Go to the project directory

```bash
cd moonlight
```

Install dependencies

```bash
flutter pub get
```

Start the server

```bash
flutter run
```

## Deployment For Android

To deploy this project run

```bash
  flutter build apk
```
